### Equivalent Capacitance in Series and Parallel

| Resistors in Series                                                                                    | Resistors in Parallel                                              |
| ------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------ |
| Current only has one path to travel                                                                    | Current has multiple paths it could travel                         |
| Current is the same for all resistors on the path                                                      | Potential difference is the same across all resistors in the combo |
| More Obstacles on the path = more resistance                                                           | More paths = less resistance                                       |
| **Capacitors in Series**                                                                               | **Capacitors in parallel**                                         |
| Current only has one path to travel                                                                    | Current has multiple paths it could travel - Different charge      |
| Charge is the same for all capacitors on the path & current is the same for all capacitors on the path | Potential difference is the same across all capacitors             |
| More capacitors on the path = LESS Capacitance                                                         | More paths = More capacitance                                      |
|                                                                                                        | Equivalent capacitance is the sum of all capacitors in parallel    |

# Equations
____________
#### Individual Capacitors 
$$Q = C \Delta V$$
#### Voltage across Capacitor
$$V_C = \frac{Q}{C}$$
- *Voltage Across a Charging Capacitor* $$V_c(t)=V_b(1-e^{\frac{-t}{RC}}$$
- *Charge on a Charging Capacitor* $$Q(t)=CV_b(1-e^{\frac{-t}{RC}})$$
- *Voltage across the resistor when Charging capacitor*$$V_r(t)=V_be^{\frac{-t}{RC}}$$
- *Current through resistor when Charging Capacitor*$$I(t)=\frac{V_b}{R}e^{\frac{-t}{RC}}$$
- *Voltage across a discharging capacitor* $$V_c(t)=V_be^{\frac{-t}{RC}}$$
- *Charge on a Discharging capacitor* $$Q(t)=Q_0e^{\frac{-t}{RC}}$$
- *Voltage across the resistor when Discharging capacitor*$$V_R(t)=V_be^{\frac{-t}{RC}}$$
- *Current through resistor when discharging capacitor*$$I(t)=-\frac{V_b}{R}e^{\frac{-t}{RC}}$$
#### Capacitors in Series
$$\frac{1}{C \tiny eq,series \normalsize}= \varSigma \frac{1}{C_n} = \frac{1}{C_1} + \frac{1}{C_2} + \frac{1}{C_2} ...$$
#### Capacitors in Parallel
$$C \tiny eq,parallel \normalsize = \varSigma C_n = C_1 + C_2 + C_3 ... C_n$$
